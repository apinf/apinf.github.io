Simple API Management - Easy API Discovery

Apinf.io platform offers a comprehensive, yet easy to use API management tool. Apinf provides simplified workflow for API owners for common Api management tasks. At the same time, it serves as a self-service tool for API consumers for discovering APIs and taking them to use.

Open source, open development

Apinf team has chosen complete openness: we have a MIT license for entire source code. There are no closed-source components. Openness extends to the software development processes: we provide full transparency and visibility to our processes, progress and documentation.

We also want to welcome all interested contributors to join the development effort.

How to join

  • There are several ways you can contribute:
  • Contribute to our planning documents
  • Report bugs on our issue tracker
